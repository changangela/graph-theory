Left click on canvas to create vertex. Left click on vertex to select and drag. To add an edge, select a source vertex, hold down SHIFT, and select a destination vertex. To delete a vertex or edge, select it and press DELETE (or right click on a vertex or edge). To edit the weight of an edge, select the edge and press ENTER. To restart, press F5.

Modes: builder (n), bipartite (b), degrees (d), eulerian (e), coloring (c), dijsktras shortest path (select source vertex, hold s, select destination vertex), minimum spanning tree (m), hamiltonian (h)

Toggles: weighted (w), labels (l)
